
# Sprint_5

Этот проект содержит набор UI автотестов для веб-приложения **Stellar Burgers**.  
Тесты покрывают функционал регистрации, авторизации, личного кабинета и раздела «Конструктор».

Установить зависимости:
pip install -r requirements.txt

Запуск тестов:
pytest -v

#Покрытые тесты
Авторизация:

Вход с корректными данными

Переходы в форму логина из разных мест

Восстановление пароля

Регистрация:

Успешная регистрация

Ошибки при некорректном пароле

Личный кабинет:

Переход в профиль

Переход из профиля в конструктор

Выход из аккаунта

Конструктор:

Переход на раздел «Булки»

Переход на раздел «Соусы»

Переход на раздел «Начинки»
